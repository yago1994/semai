Safari extension for Outlook Web

Welcome to remou

Draft, refine, and send Outlook Web replies with the same accent language as the extension itself.

You can turn on remou’s extension in Safari Extensions preferences.

Get started

  1. Open Safari settings and enable the remou extension.
  2. Open Outlook on the web in Safari and select an email thread.
  3. Use the remou panel inside Outlook to generate, insert, or send a Reply all draft.

What to expect

  • remou works on Outlook Web pages inside Safari.
  • When the extension is enabled, its tools appear in the Outlook interface.
  • If you do not see remou, refresh Outlook after turning the extension on.
